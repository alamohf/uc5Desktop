# uc5Desktop

Projeto integrador da unidade 5 - aplicacao desktop com Electron + TypeScript.

## Requisitos

- Node.js 18 ou superior instalado

## Como rodar

```bash
git clone https://github.com/alamohf/uc5Desktop.git
cd uc5Desktop
npm install
npm start
```

O comando `npm start` compila o TypeScript (`npm run build`) e abre a aplicacao Electron.

## Scripts

- `npm run build` - compila o codigo TypeScript de `src/` para `lib/`
- `npm start` - compila e abre a aplicacao
- `npm run lint` - roda o tslint sobre o codigo em `src/`
